# Metodos_de_Busqueda
-- Estado del proyecto: Finalizado.

  ## Descripción
Este proyecto contiene la implementación de diferentes métodos de búsqueda aplicados a problemas reales, utilizando el lenguaje Python.
Se desarrollaron ejemplos prácticos para comprender cómo funcionan los algoritmos de búsqueda y en qué situaciones se utilizan.

### Métodos Implementados
1. Búsqueda Secuencial

Se utiliza para recorrer una lista elemento por elemento hasta encontrar el dato buscado.

* Ejemplo: Verificar asistencia de estudiantes o Revisar productos agotados

2. Búsqueda Binaria

Se aplica en listas ordenadas y permite buscar de forma más rápida dividiendo el conjunto de datos.

* Ejemplo: Buscar una calificación o Validar códigos de descuento

3. Búsqueda Externa

Se usa para manejar grandes volúmenes de datos, como registros almacenados en archivos o bases de datos.

* Ejemplo: Buscar un cliente por cédula en una lista grande
